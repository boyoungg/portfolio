OVERVIEW
프로젝트 스튜디오 수업에서 기획, 디자인, 프로그래밍의 총 3명의 팀원으로 진행한 세대 커넥트 서비스 SEE : SAW의 웹페이지 작업을 진행하였습니다.
팀장 및 개발자의 포지션으로 참여하였고 ONE SCROLL 형태의 웹으로 섹션 별로 이동할 수 있는 HEADER와 중간중간 SLIDE INTERACTION을
추가하여 서비스의 정보를 소개하는 페이지 입니다.

DEVELOPMENT

MAIN PAGE (ONE SCROLL)
기획 참여도 50%
디자인 참여도 40%
프로그래밍 참여도 100%

LANGUAGE

HTML / CSS / JAVASCRIPT
JQUERY / CHART JS / SLICK JS

DEVICE

PC

DEVELOP POINT
CLICK EVENT :메인화면의 앱 화면을 클릭하면 앱화면이 사라지고 GIF 목업 등장
SLICK JS : SLICK SLIDE를 이용하여 캐릭터 소개 페이지 슬라이드 제작
SETINTERVAL : 시간별로 배경이미지가 무한반복되며 변경되는 색상 바 제작

CODE REVIEW

SETINTERVAL COLOR BAR

SETINTERVAL을 활용하여 TOGGLE 되는 배경 색상바를 작업하였습니다
DIV의 색상은 TOGGLE( ) 함수 안에 들어있는 설정값으로 무한반복되고
색상이 변화할때 TRANSITION이 함께 변화하는 것을 알 수 있습니다
