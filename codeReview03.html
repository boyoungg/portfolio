JEJU TRAVEL MAP _ 제주, 놀멍놀멍 봅서!
2022. 04. 07 ~ 2022. 04. 21
OVERVIEW
2학년 1학기 웹퍼블리싱 수업에 제주 여행을 주제로 작업한 제주도 관광지, 향토 음식등을 소개하는 페이지 입니다.
제주도가 컨셉인 일러스트들을 활용하여 디자인 하였고 제주도의 방언을 활용하여 레이블링 및 내용 작성을 하였습니다.
여행의 설램을 담은 여행지도를 꾸밀 수 있는 부분이 있으면 좋을 것 같아 제주 아이콘들을 디자인하였고 이를 바탕으로 제주여행지도를 꾸밀 수 있습니다.

DEVELOP POINT
HEADER MOUSE EVENT :HEADER 영역에 마우스가 HOVER 되면 마우스 인터렉션이 발생
MOUSE -> FRAME EVENT :마우스의 값을 이용하여 마우스 움직임에 따라 Rotate값이 변화
JQUERY UI :아이콘들을 DRAG & DROP 되게 제작하여 제주 지도를 스티커북 처럼 꾸밀 수 있습니다

CODE REVIEW

JQUERY UI

JQUERY 에서 제공하는 JQUERY UI 라이브러리를 이용하여 아이콘이
DRAG & DROP 되는 기능을 추가하였습니다.

오른쪽 DIV 안에 있는 이미지들을 드레그 하여 지도 위에 두고 스티커 북
처럼 자신이 웹을 보며 가고싶었던 장소 또는 하고싶었던 부분들을

표시해둘 수 있는 목적으로 작업한 페이지 입니다.
