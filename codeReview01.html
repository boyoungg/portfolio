PORTFOLIO _ WAVE(파동)
2022. 11. 30 ~ 2022. 12. 14
OVERVIEW
1학년 1학기 디자인 기초 수업에서 진행한 포트폴리오 주제의 ‘WAVE’를 다시 활용하여 새로운 방향성을 제시하고 리디자인을 진행하여 제작한
웹 퍼블리싱 포트폴리오입니다. 해당 포트폴리오는 파동(WAVE)이 변화하는 모습을 시각적인 디자인요소로로 활용하여 디자인 하였고
현재의 나의 모습, 활동, 경험을 각각 세가지의 파동으로 분류하여 이를 바탕으로 인터렉션과 전체적인 포트폴리오의 스토리텔링을 진행하여 제작하였습니다

CODE REVIEW

WAVE INTERACTION

작업물 제작시기에 따라 제작 능력을 WAVE 안 흔들리는 물로 시각화 하여
표현한 CSS 애니메이션 입니다 WAVE 왼쪽의 버튼을 클릭했을 때 변수로
지정한 i의 값이 1씩 증가하게 되며 해당하는 CSS 애니메이션이 추가됩니다

버튼이 CLICK 되면 IF문이 실행되고 변수 i의 값이 변함에 따라
각각 해당하는 if 의 결과가 출력됩니다.

해당하는 내용은 JQUERY 의 HTML( ) , ADDCLASS( ), REMOVECLASS( ), ATTR( ) 함수를 활용하여 내용 변경을 해주었습니다
